# Features Overview

Taskin provides powerful features for managing daily tasks and routines.

## Core Features

### Dependency Management
Define complex relationships between tasks and categories. Tasks become available only when their dependencies are satisfied.

[Learn more →](dependency-graph.md)


### Reset Intervals
Configure automatic task resets on daily, weekly, or custom intervals.

[Learn about reset intervals →](../configuration/reset-intervals.md)

### Reports & Statistics
Track completion rates, skip rates, and time spent on tasks over time.

[View reports →](reports.md)

### Dependency Graph Visualization
Interactive Mermaid-based graph showing your entire task structure with color-coding and zoom controls.

[Explore the graph →](dependency-graph.md)

### One-off Tasks
Create temporary, ad-hoc tasks that don't fit into your regular routine.

[Manage one-offs →](../configuration/oneoff-tasks.md)
